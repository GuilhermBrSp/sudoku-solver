# Algoritmos Solucionadores de Sudoku

## Descrição
Projeto que implementa algoritmos solucionadores do jogo Sudoku em forma de NoteBooks.
Este projeto faz parte de um seminário apresentado no segundo semestre de 2020, na matéria de Inteligência Artificial da Fatec São Paulo.

## Algoritmos até então implementados
- **BackTracking Recursivo**: Algoritmo de força bruta implementado utilizando backtracking recursivo
- **Satisfação de Restrições (CSP)**: Algoritmo implementado na forma de um problema de satisfação de restrições (constraint satisfaction problem), utilizando a biblioteca [python-constraint](https://pypi.org/project/python-constraint/)
